## About the Project

it's a simple User list Page, You can add news users, search users by name, e-mail or city and sort the users.

## Build With

- Vite with React
- Redux
- Antd
- Axios

## Getting Started

You need Node Version +16.

Install the dependences

```sh
npm install
```

run the project

```sh
npm run dev
```
